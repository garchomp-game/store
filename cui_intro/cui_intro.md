# CUIの初歩
## GUIとCUI
- GUI(Graphical User Interface)
  - 絵付きの画面
  - マウスでカチカチして操作
  - 直感的
- CUI(Character User Interface)
  - 文字のみの画面
  - 所謂黒い画面
  - コマンドで操作
  - 出力結果も全部文字のみ

## cmd
- 起動方法  
`Win+R→cmd→Enter`が一番モテるやり方
- (ちょっと違うけど、)cmdはExplorerの、CUI版のようなもの  
厳密にいうと、  
```
cmdはCUIツールであり、ExplorerのCUI版としての機能も兼ねてる
```
という感じ
